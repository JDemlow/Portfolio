What I do

HTML + CSS + JavaScript

My goal with any project is to use semantic HTML to add meaning and accessibility to your site. I am able to create beautiful and vibrant web pages by using CSS frameworks such as Tailwind, Bootstrap, and CSS pre-processors such as Sass. JavaScript then comes in to create an interactive and engaging experience for your site users. This trifecta is the core of any great website for businesses, and the foundation for how your users experience your presence on the web.

Responsive Web Design

Modern web design should be completely responsive. That is to say that your site will appear fantastic on desktop, tablet, mobile, or any other screen you can think of. By incorporating Flexbox and Grid as part of your site creation, you can be sure that users everywhere and at every screen size will have the experience you want and see everything you have to offer them.

Search Engine Optimization

It's always a major priority of mine that users are able to find your site quickly and easily. SEO is a critical part of any site, and yours should be no exception. Making sure your site loads quickly, is responsive on all devices, adding meta tags, controlling image size, and so much more goes into making your site show up when users search for you. I will always make sure your site is optimized prior to delivery.
